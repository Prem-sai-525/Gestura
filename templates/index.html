Gesture Control System

Status: Stopped

How to Use Gestures:

  • Play: Index + Pinky up, other fingers closed
  • Pause: Thumb + Index + Pinky up, others closed
  • Forward: Thumb pointing right, others closed
  • Backward: Thumb pointing left, others closed
  • Volume Up: Thumb + Index + Middle + Pinky up, ring closed
  • Volume Down: Index + Middle up, others closed
  • Mute/Unmute: Thumb + Pinky wave, others closed
Note: Play any video to use Forward & Backward gestures. Other gestures work even if no video is playing.
